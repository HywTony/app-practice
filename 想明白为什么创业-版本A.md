# 想明白为什么创业，先问自己这几个问题

## 正文内容

想明白为什么创业，先问自己1个核心问题

你是以做事的逻辑去赚钱？
还是以赚钱的逻辑去做事？

做事 → 赚钱 = 可能成功
赚钱 → 做事 = 永远失败

很多人辞职创业，以为换个身份就能逆袭
但创的好是老板，创的不好比牛马还牛马

记住段永平老师的话：
十年二十年后回头看，有些朋友永远在原地打转

而你呢？到底是在做事还是在赚钱？

----

"做对的事情，把事情做对"

这句话说得够深了
做事需要思维模式，赚钱也需要思维模式
但大多数创业者缺的不是机会，是思维

社会的金字塔是铁律
不断学习和进步，才是最终的出口

----

那怎么判断自己是哪种？

看这3个问题👇

1️⃣ 你做的事，3年后还有价值吗？
   如果只是赚快钱，3年后可能就过时了

2️⃣ 你能讲清楚自己在解决什么问题吗？
   做事的人能说清，赚钱的人说不清

3️⃣ 你每天想的是"怎么做好"还是"怎么赚快钱"？
   前者关注长期，后者只看眼前

如果3个问题都答不上来
说明你还没想明白

想明白了再创业，不迟
