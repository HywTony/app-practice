# 🚀 巨爆铺小红书自动化营销 - 快速开始指南

## 📋 系统概述

这是一套完整的小红书自动化营销解决方案，可以：

1. **自动生成内容** - 使用AI生成高质量小红书笔记
2. **智能标签匹配** - 自动选择合适的话题标签
3. **定时发布** - 按计划自动发布内容
4. **效果追踪** - 记录发布日志，追踪效果

---

## 🎯 10分钟快速上手

### 步骤1：准备工作（5分钟）

#### 1.1 设置API密钥

获取Claude API密钥（访问 https://console.anthropic.com）

**Windows用户：**
```cmd
set ANTHROPIC_API_KEY=sk-ant-xxxxx
```

**Mac/Linux用户：**
```bash
export ANTHROPIC_API_KEY=sk-ant-xxxxx
```

#### 1.2 安装依赖

```bash
cd xiaohongshu_automation
pip install -r requirements.txt
```

### 步骤2：第一次生成内容（2分钟）

```bash
python quick_start.py
```

选择 `1. 生成一篇测试内容`

你会看到类似这样的输出：
```
📝 正在生成内容...
内容类型: 工具合集型
模板: 工具合集-效率提升

✅ 内容生成成功！

标题: Temu卖家必备！这5个功能让效率飙升10倍！

正文预览:
做了半年Temu，终于找到宝藏工具了！🎉...

话题标签: #跨境电商运营 #TEMU运营工具 #电商效率神器
```

### 步骤3：查看和发布（3分钟）

#### 3.1 查看生成的内容

选择 `3. 查看最新生成的内容`

#### 3.2 保存为草稿

选择 `4. 发布最新内容（保存为草稿）`

内容会保存在 `logs/draft_*.json`

#### 3.3 手动发布到小红书

**方式A：使用Claude Code（推荐）**

在Claude Code中运行：
```python
# 读取草稿
import json
with open('xiaohongshu_automation/logs/draft_20250101_120000.json') as f:
    draft = json.load(f)

# 使用MCP发布
mcp__xiaohongshu_mcp__publish_content(
    title=draft['title'],
    content=draft['content'],
    images=draft['images']  # 记得替换为真实图片路径
)
```

**方式B：手动复制粘贴**

1. 打开草稿文件
2. 复制标题和内容
3. 在小红书APP中发布

## 📚 详细使用教程

### 1️⃣ 生成内容

#### 单篇生成

```bash
cd src
python content_generator.py --test
```

#### 批量生成

```bash
python content_generator.py --count 5
```

生成的内容保存在 `logs/content_*.json`

### 2️⃣ 自定义配置

编辑 `config/config.yaml`：

```yaml
# 修改发布时间
content_strategy:
  post_times:
    - "08:00"  # 改为你想要的时间
    - "19:00"

# 修改话题标签
hashtags:
  primary:
    - "你的主要标签"
```

### 3️⃣ 启动自动化

#### 测试定时任务

```bash
cd src
python scheduler.py --test
```

#### 启动调度器

```bash
python scheduler.py --start
```

调度器会按照配置的时间自动生成内容。

**公式1：数字+利益点**

## ⚠️ 常见问题

### Q1: API调用失败

**问题：** `AuthenticationError: Invalid API key`

**解决：**
```bash
# 检查环境变量
echo $ANTHROPIC_API_KEY  # Linux/Mac
echo %ANTHROPIC_API_KEY%  # Windows

# 重新设置
export ANTHROPIC_API_KEY=your-key
```

### Q2: 生成的内容重复

**问题：** 内容相似度太高

**解决：**
1. 增加模板数量（编辑 `config/templates.json`）
2. 调高AI temperature（`config.yaml` 中设置为 0.9）
3. 手动审核并修改

### Q3: 无法发布到小红书

**问题：** 发布失败或权限不足

**解决：**
- 使用草稿功能，手动复制到小红书APP
- 在Claude Code环境中使用MCP工具
- 确保小红书账号已登录

### Q4: 图片占位符

**问题：** 生成的是占位图片

**解决：**
1. 使用Canva设计真实图片
2. 使用AI生成图片
3. 准备产品截图和场景图
4. 替换 `assets/images/` 中的图片

### Q5: 定时任务不执行

**问题：** 到了设定时间没有自动执行

**解决：**
```bash
# 检查配置
cat config/config.yaml

# 测试调度器
python scheduler.py --test

# 确认系统时间正确
date  # Linux/Mac
time  # Windows
```
